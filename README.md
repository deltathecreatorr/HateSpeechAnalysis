# HateSpeechAnalysis
This project will investigate the link between real world kinetic events, and the effect it has on the hate speech and derogatory comments made on various social media platforms. This is a text/network analysis project.

Install the packages of this project using

```
pip install -r requirements.txt
```

The Twitter dataset was scraped using twikit, but the dataset is saved, so it is unneccesary to do it again. 

The Reddit dataset is from [Kaggle](https://www.kaggle.com/datasets/asaniczka/reddit-on-israel-palestine-daily-updated)

The network script can be run using the command
```
python src/build_network.py
```
Assuming you are in the root folder of this project when cloned.
